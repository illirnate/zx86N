Welcome to zx86.org!

This is a website built for old internet browsers & old or new systems alike!

Why?

Well, I always wanted to make an old 2000's, maybe 2010's styled website. The reason as to why zx86.org automatically goes to a http:// link, is well, for compatibility reasons. Most old web browsers, like Internet Exploder for Microsoft Windows 95, (Fuck you Micro$oft, I'm still not going to use Edge.) don't really support https, since it wasn't primarily adopted during that time.

Its still fun to have an old-styled website.

What's in the downloads?

Downloads for old operating systems, or even whole operating systems, drivers, random files, archives, a bunch more random files, applications aaaaand... yea that's basically it. There are going to be more categories soon, but as of today (05/05/2024), those are it for now. Adding more files soon!!
